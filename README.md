# Aliza Habib — Portfolio

Personal portfolio site, built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Content

All case study and profile content lives in `src/lib/data.ts` — a single source of truth for the case studies, clients, testimonials, and profile info shown across the site.

## Deployment

Deployed on Vercel, auto-deploying from the `main` branch.
